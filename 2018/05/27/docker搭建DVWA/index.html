docker搭建DVWA

发表于 2018-05-27 | 分类于 docker

1. 先创建一个新的容器

在创建容器时，会自动获取一个容器id，对于每个容器他的id都是唯一的

2. 进入正在运行的容器

sudo docker exec -it id /bin/bash
id是创建容器时自动获取的，可以通过id对容器进行操作，也可以通过容器的名字操作容器

3. Lamp（web应用软件组合）

apache

1.安装apache
apt-get install apache2

2.启动apache
service apache2 start

3.测试apache

MySQL

安装mysql
apt-get install mysql-server mysql-client
mysql-server是服务，mysql-client通过模块操作数据库

PHP（一种脚本语言）

1.安装PHP和一些包
apt-get install php7.2
因为Ubuntu是自带PHP的相关文件的，建议安装的时候看一下自带的php的相关文件的版本，然后直接安装那个版本比较方便，然后我用的Ubuntu是18.04，php是7.2

2.测试php
在/var/www/html这个目录下我写了一个phpinfo();的1.php文件来测试php是否正常，然后重新启动一下apache，然后通过浏览器访问了一下1.php

3.访问1.php页面正常显示

4. 下载dvwa并部署到apache

用到的一些命令需要安装，如wget，unzip

1.下载的dvwa

2.部署到apache

3.通过浏览器访问

5.配置完成后dvwa
